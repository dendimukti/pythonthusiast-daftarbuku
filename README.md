## Menggunakan Git dgn Command Line
- Create SSH Keys
- Upload Ke Github
- Buat Repo
- Copy Alamat SSH
- Clone dgn perintah git clone <alamat>
- Ubah File
- Tambahkan Ke Daftar file yg di ubah dengan git Add
- Simpan perubahan dengan git commit
- kirim perubahan ke github dengan git push
